# stock-guide-agent
Multi-agent AI stock guide system using Google ADK,YFinance,News API .
This agent is designed to provide comprehensive educational insights rather than direct investment advice. The system helps users understand concepts such as SIPs, stock market fundamentals, current market trends, company status, and future outlooks through AI-driven analysis and explanations.
